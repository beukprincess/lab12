Що таке GNU/Linux?

GNU/Linux — це вільна та відкрита операційна система, що складається з:

  • ядра Linux, яке розробив Лінус Торвальдс у 1991 році;
  • компонентів GNU, які надають необхідні утиліти та бібліотеки.

Це альтернатива комерційним ОС, яка використовується на серверах, ПК, мобільних пристроях (Android) тощо.

Переваги GNU/Linux:

  • ✅ Відкритий код
  • ✅ Безкоштовність
  • ✅ Висока безпека
  • ✅ Гнучкість та налаштування
  • ✅ Стабільність

Найпопулярніші дистрибутиви Linux (станом на 2025):

Дистрибутив Основні особливості
Ubuntu Простість у використанні, велика спільнота, популярний серед новачків
Debian Надійність, стабільність, часто використовується на серверах
Fedora Новітні технології, підтримується Red Hat
Linux Mint Зручний для переходу з Windows, базується на Ubuntu
Arch Linux Для досвідчених користувачів, мінімалізм, філософія KISS
Manjaro Базується на Arch, але більш зручний для користувачів
openSUSE Стабільність, інструменти для розробників та адміністраторів
Pop!_OS Розроблений System76, оптимізований для інженерів та геймерів
Kali Linux Для тестування безпеки, використовується етичними хакерами
CentOS Stream Базується на Red Hat, підходить для серверів
